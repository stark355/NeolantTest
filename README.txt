Управление камерой - WASD + мышь при зажатом колесике
Первое задание - выбранный объект изменяет цвет. Нельзя удалить или изменить невыбранный объект.
Третье задание - материал меняется от черного к белому и обратно по нажатию левой и правой клавиши мыши соответственно.

Четвертое задание - задается размер массива (по умолчанию - от 1 до 15 включительно) и выбирается конкретный префаб
из списка префабов по клику на имя необходимого префаба (под кнопкой Generate)
Для изменения размера объекта необходимо зажать ПРОБЕЛ и управлять размером с помощью направляющих
Для поворота вокруг требуемой оси необходимо зажать LeftControl и управлять поворотом с помощью направляющих
(Внимание! Для поворота вокруг оси X необходимо двигать мышь вверх и вниз, для поворота вокруг осей Y и Z - влево и вправо)
